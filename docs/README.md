## ADR

Chưa có ADR formal. Khi cần record kiến trúc decision, dùng template:

```
# ADR-NNNN: <decision-title>
Date: YYYY-MM-DD
Status: Proposed | Accepted | Superseded by ADR-XXXX

## Context
<vấn đề + ràng buộc>

## Decision
<chốt như thế nào>

## Consequences
<hệ quả tích cực / tiêu cực / trung tính>
```

Đặt file `adr/NNNN-short-slug.md`.
